Test Role-Based Routing System

1. Routing Logic Overview

Source Column:
- Admin/Bendahara/Ketua: dari kolom jabatan
- Siswa: dari kolom sebagai

Unit YPWILUTIM (Pusat):
- jabatan contains 'admin/operator/tu/administrasi' → /admin/pusat
- jabatan contains 'bendahara/keuangan/finance' → /bendahara/pusat
- jabatan contains 'ketua/kepala/pimpinan/leader' → /ketua/pusat
- jabatan lainnya → /yayasan

Unit Lain (Sekolah):
- jabatan contains 'admin/operator/tu/administrasi' → /admin/sekolah
- jabatan contains 'bendahara/keuangan/finance' → /bendahara/sekolah
- jabatan contains 'ketua/kepala/pimpinan/leader' → /ketua/sekolah
- jabatan lainnya → /guru

Siswa:
- sebagai = 'siswa' → /student (semua unit)
